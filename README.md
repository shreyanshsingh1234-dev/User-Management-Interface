# 🚀 Nexus Dashboard

A modern, high-performance User Management Dashboard built with **React** and **Vite**. This project features a premium "Glassmorphism" aesthetic, real-time API integration, and full CRUD capabilities.

Link to the website go and check it out!:-https://user-management-interface-kappa.vercel.app

![Nexus Dashboard Preview](/Users/admin/.gemini/antigravity/brain/f197bb5d-0059-4884-88b2-f511d6c31ef1/dashboard_user_list_clean_1770886415871.png)

## ✨ Key Features

- 🔐 **Secure Authentication**: Fake auth flow with persistence using LocalStorage.
- 👥 **User Management**: Add, Edit, and Delete users with real-time state updates.
- 🔍 **Dynamic Search**: Filter users instantly by name, email, or username.
- 📡 **API Integration**: Data fetched from JSONPlaceholder for a realistic experience.
- 🎨 **Premium UI/UX**:
  - Glassmorphism design system.
  - Smooth animations and transitions.
  - Fully responsive for mobile, tablet, and desktop.
- ✅ **Form Validation**: Comprehensive client-side validation for all user inputs.

## 🛠️ Built With

- **React 19** - UI Component Library
- **React Router 7** - Declarative Routing
- **Vite** - Lightning-fast Build Tool
- **Vanilla CSS** - Custom Design System (No utility frameworks)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd login-page-project
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
